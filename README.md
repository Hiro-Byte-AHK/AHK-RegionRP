# AHK-RegionRP

> RegionRP Assistant – универсальный помощник для сотрудников правоохранительных органов
> Версия 2.1b

RegionRP Assistant – это мощный инструмент для сотрудников полиции, ГИБДД, следователей и других государственных структур в RP-проектах. Программа предоставляет быстрый доступ к полным текстам законов, позволяет формировать список вменяемых статей, автоматически рассчитывает суммарное наказание и содержит множество полезных функций для повседневной службы.Работает на основе AutoHotkey v2.0

📚 Быстрый поиск по 8 кодексам и уставам (УК, КоАП, УПК, ПДД, Устав ГИБДД, ФЗ «О полиции», ФЗ «О государственной собственности», Трудовой кодекс, Устав ГУ МВД).

➕ Формирование списка обвинения – двойной клик по статье добавляет её в «список вменяемых» с полным описанием и наказанием.

⚖️ Автоматический расчёт суммарного срока – программа показывает общий срок лишения свободы (по УК) или административного ареста (по КоАП).

🖱️ Удаление статей – правым кликом по статье в любом окне можно убрать её из списка.

🔍 Живой поиск – фильтрация статей по номеру или названию.

🗺️ Полупрозрачная карта (F9) – наложение карты местности поверх игры (требуется файл map.png).

📜 Окно прав задержанного (F4) – готовый текст для зачитывания прав (аналог «прав Миранды»).

⚡ Биндеры быстрых фраз – настраиваемый текст для отправки в чат по горячим клавишам (Alt+1 – жетон, Alt+2 – произвольная фраза, Alt+X – часы).

🔄 Автоматическая проверка обновлений – при старте сверяет версию с GitHub и предлагает скачать новую.

> Вкладка «Описание» – список горячих клавиш, инструкция по использованию, контакты разработчиков.
> 
> Вкладка «Настройки» – поля для редактирования:
> 
> Текст для Alt+1 (жетон) – можно вводить как с /do, так и без него.
> 
> Текст для Alt+2 (биндер) – поддерживает префиксы /me, /do, /todo.
> 
> Все настройки сохраняются в файле settings.ini рядом с программой.
